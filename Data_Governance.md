# 命名规范
| 参数名称   | 参数描述                                     | 算法说明                                                     |
| ---------- | -------------------------------------------- | ------------------------------------------------------------ |
| date       | 交易所行情日期                               |                                                              |
| code       | 证券代码                                     |                                                              |
| open       | 开盘价                                       |                                                              |
| high       | 最高价                                       |                                                              |
| low        | 最低价                                       |                                                              |
| close      | 收盘价                                       |                                                              |
| preclose   | 前收盘价                                     | 见表格下方详细说明                                           |
| volume     | 成交量（累计  单位：股）                     |                                                              |
| amount     | 成交额（单位：人民币元）                     |                                                              |
| adjustflag | 复权状态(1：后复权，  2：前复权，3：不复权） |                                                              |
| turn       | 换手率                                       | [指定交易日的成交量(股)/指定交易日的股票的流通股总股数(股)]*100% |
